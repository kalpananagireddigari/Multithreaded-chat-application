**Multithreaded Chat Application :**

This project is a Java-based Multithreaded Chat Application that enables multiple clients to communicate simultaneously with a server using socket programming. 

The system is designed using multithreading concepts, where each client connection is handled by a separate thread, allowing real-time concurrent communication.

The application demonstrates the practical implementation of:

Java Socket Programming.

Multithreading (Thread / Runnable).

Client-Server Architecture.

Concurrent Communication Handling
